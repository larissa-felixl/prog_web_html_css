Guia de Tags HTML

Tags Mais Utilizadas

Confira algumas das tags HTML mais comuns e seus exemplos de uso.

<!DOCTYPE> - Declaração do Tipo de Documento

<!DOCTYPE html>

<html> - Estrutura do Documento

<html> ... </html>

<head> - Cabeçalho do Documento

<head> ... </head>

<meta> - Metadados

<meta charset="UTF-8">

<title> - Título da Página

<title>Meu Site</title>

<body> - Corpo da Página

<body> ... </body>

<h1> até <h6> - Títulos

<h1>Título Principal</h1>

<p> - Parágrafo

<p>Texto de exemplo.</p>

<a> - Link

<a href="https://www.exemplo.com">Visite o Exemplo</a>

<img> - Imagem

<img src="imagem.jpg" alt="Descrição da imagem">

<div> - Container Genérico

<div>Conteúdo aqui</div>

<span> - Inline Container

<span>Texto em linha</span>

<ul> & <li> - Lista Não Ordenada

<ul> <li>Item</li> </ul>

<ol> & <li> - Lista Ordenada

<ol> <li>Item 1</li> </ol>

<table> - Tabela

<table><tr><td>Dado</td></tr></table>

<form> - Formulário

<form> <input type="text"> </form>

<input> - Entrada de Dados

<input type="text" placeholder="Digite aqui">

<button> - Botão

<button>Clique Aqui</button>

<label> - Rótulo

<label for="nome">Nome:</label>

<textarea> - Área de Texto

<textarea>Digite aqui</textarea>

<select> & <option> - Caixa de Seleção

<select><option>Opção 1</option></select>

<iframe> - Incorporar Conteúdo

<iframe src="https://www.exemplo.com"></iframe>

<script> - Scripts

<script>alert('Olá!');</script>

<link> - Importar Estilos

<link rel="stylesheet" href="style.css">

<style> - Estilos CSS

<style>body {color: white;}</style>
